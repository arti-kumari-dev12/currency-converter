# Currency Converter Web App 💱

A simple and lightweight currency converter web application that allows users to convert currencies instantly using real-time exchange rates. It also keeps a record of previous conversions for easy reference.

## Features
- Real-time currency conversion using API  
- Swap currency option for convenience  
- Conversion history tracking  
- Clean and user-friendly interface  

## Tech Stack
- HTML  
- CSS  
- JavaScript  
- REST API  

## Project Purpose
This project was built to practice frontend development, API integration, and basic state handling in web applications.

## Note
This is a frontend-focused project with API integration for live exchange rates.
